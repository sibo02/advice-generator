# Advice Generator App

A simple web app that generates random advice with the click of a button.

## How It Works
Click the dice button to get a new piece of advice fetched from the Advice Slip API in real time.

## Built With
- HTML5
- CSS3
- JavaScript
- Advice Slip API

## What I Learned
- How to fetch data from a real API using JavaScript
- How to update the DOM dynamically
- Responsive design with CSS media queries
